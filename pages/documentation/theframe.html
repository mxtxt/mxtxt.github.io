The frame wraps the actual content and contains three menus: for the navigation through mxTXT pages, the selection of a document-style combination, and the execution of the random or the source button. All the menus and the application of mxTXT styles is based on JavaScript. The implementation is explained in the following.

iFrame

The iFrame technology was employed in order to have no limitations in the styling process. An iFrame allows opening an HTML file within another HTML file without any interference of the two. This trick allows the full separation of mxTXT styles from the cascading style sheets (CSS) used for the frame. Without using iFrame, it would have been necessary to consider the HTML and CSS of the mxTXT user interface while creating the styles.

The iFrame is implemented with the following HTML markup and CSS styling:

                  
  <div id="content">
    <iframe id="frameDocument" src="pages/home.html"></iframe>
  </div>
                  
                  
  #content {
    width: 100vw;
    height: 100vh;
    overflow: scroll!important;
    -webkit-overflow-scrolling: touch!important;
  }
  #content iframe {
    position: absolute;
    top: 2.5rem;
    left: 2.5rem;
    width: calc(100vw - 5rem);
    height: calc(100vh - 5rem);
    border: none;
  }
  @media (max-width: 767px) {
    #content iframe {
      top: 0;
      left: 0;
      width: 100vw;
      height: 100vh;
    }
  }
