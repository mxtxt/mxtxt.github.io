User Guide

Want to use mxtxt on your own documents? Here's some important things to keep in mind.

Format

If you want to use mxtxt on a personal file, make sure that it is in html format and it includes the relevant declaration:

                
                  <!DOCTYPE html>

General Structure

Your HTML document should follow a logical structure evidenced in the markup. As such, it should look like this:

                
  <html>
    <head> <!--- in the head you put all your metadata. --->
      <meta charset="utf-8"> <!--- in the meta tag you specify the character set used in the page. --->
      <link name="theme" rel="stylesheet" type="text/css" href="#"> <!--- link is used for linking the stylesheet. --->
    </head>

    <body> <!--- the body contains all the actual content of the document. --->
      <header> <!--- the header defines a header section for the element that contains it. --->
        <h1>Your magazine title. </h1> <!--- title of the document. --->
        <h2>document title. </h2> <!--- title of the content of the document. --->
        <h3>document subtitle. </h3> <!--- subtitle of the document. --->
        <p>You can add a paragraph to your header. </p>
        <!--- p defines a paragraph of a text.
        You can put the name of the author/illustrator in the p within the header. --->
      </header>

      <main>
        <!--- The main tag specifies the main content of a document. The content inside the main element should be unique to the document.
        It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search forms.
        Note: There must not be more than one main element in a document. The main element must not be a descendant of an article, aside, footer, header, or nav element. --->

        <section> <!--- used if the document contains clearly separate sections or chapters. --->
          <h4>This is the section title</h4> <!--- section title. --->
          <p> This is the first line. <br> <!--- br is used for line breaks, like in dialogues. ---->
          This is the <em>second</em> line. </p> <!--- em is used for emphasized content. --->

          <p><small>The text size of this paragraph will be smaller. </small></p> <!--- small is used for smaller content. --->

          <p> This is a paragraph. <a href="#"></a></p>
          <!--- a defines a hyperlink, which is used to link from one page to another.
          href is the most important attribute of the ; element. It indicates the link's destination.
            href=“URL” link to another webpage;
            href=“#value” link to an element with a specified id (id=“value”) within a page.
          If the href attribute is not present, the a tag is not a hyperlink. --->

          <p> This is a paragraph. <a id=""></a></p>
          <!--- The id attribute can be used to define bookmarks in a page. HTML bookmarks are used to allow readers to jump to specific parts within the same Web page.
          Bookmarks can be useful if your webpage is very long. To make a bookmark, you must first create the bookmark, and then add a link to it.
          When the link is clicked, the page will scroll to the location with the bookmark. Use the href attribute (href="#value") to link the bookmark to another part of the same webpage. --->

          <q>This is a quotation. </q> <!--- q is used for short quotations--->

          <figure> <!--- figure specifies self-contained content, like illustrations, diagrams, photos, code listings, etc. --->
            <img src="../path/to/your/figure.jpg"> <!--- the src attribute of img specifies the URL of the image--->
            <figcaption> This is the figure caption. </figcaption> <!--- figcaption is used for defining a caption for a figure element. --->
          </figure>
        </section>

        <section>
          <p> This is a paragraph. </p>

          <p> This is a paragraph. </p>

          <figure>
            <video src="../path/to/your/video.jpg"> <!--- the src attribute specifies the URL of the video. --->
            <figcaption> This is the figure caption. </figcaption>
          </figure>
        </section>

        <section>
          <p> This is a paragraph. </p>

          <p> This is a paragraph. </p>

          <figure>
            <video src="../path/to/your/video.jpg">
            <figcaption> This is the figure caption. </figcaption>
          </figure>
        </section>

        <section>
          <p> This is a paragraph. </p>

          <figure>
            <table> <!--- table can have th, tr, td, colspan, rowspan and caption within it. --->
              <caption>This is the table caption. </caption> <!--- caption is used for creating table captions. --->
              <tr> <!--- tr defines a table row. --->
                <th>This is a table header</th> <!--- th defines a table header, which is bold and centered by default. --->
                <th>This is another table header</th>
              </tr>
              <tr>
                <td>These are table data in a cell</td> <!--- td defines a table cell containing data. --->
                <td>This is another cell</td>
              </tr>
            </table>
          </figure>

        </section>

        <section>

          <figure>
            <table>
              <tr>
                <td rowspan="">This is a cell</td>
                <td colspan="">This is a cell</td>
                <td></td>
                <!--- rowspan and colspan are attributes of th and td.
                They are used to specify the number of rows and columns a cell should span. --->
              </tr>
              <tr>
                <td>This is a cell</td>
                <td>This is a cell</td>
              </tr>
             </table>
          </figure>

          <ol> <!--- ol defines an ordered list, which can be numerical or alphabetical. --->
            <li>This is the first item. </li> <!--- li defines an item of the list. --->
            <li>This is the second item. </li>
            <li>This is the third item. </li>
          </ol>

          <ul> <!--- ul defines an unordered list. --->
            <li>This is one item of the list. </li>
            <li>This is another item of the list. </li>
          </ul>

        </section>
      </main>

      <footer>
        <p>Here you can put the copyrights. </p>
      </footer>
      <!--- The footer tag defines a footer for a document or section.
      Used for credits, copyrights, etc. You can have several footer elements in one document. --->

    </body>
  </html>
