User Guide

Want to use mxtxt on your own documents? Here's some important things to keep in mind.

Format

If you want to use mxtxt on a personal file, make sure that it is in html fomrat and it includes the relevant declaration:

General Structure

Your HTML document should follow a logical structure evidenced in the markup. As such, it should look like this:

                
                  <html>
                    <head> 
                      <meta charset="utf-8"> 
                      <link name="theme" rel="stylesheet" type="text/css" href="#"> 
                    </head>

                    <body> 
                      <header> 
                        <h1>Your magazine title. </h1> 
                        <h2>document title. </h2> 
                        <h3>document subtitle. </h3> 
                        <p>You can add a paragraph to your header. </p>
                        
                      </header>

                      <main>
                      

                        <section> 
                          <h4>This is the section title</h4> 
                          <p> This is the first line. <br> 
                          This is the <em>second</em> line. </p> 

                          <p><small>The text size of this paragraph will be smaller. </small></p> 

                          <p> This is a paragraph. <a href="#"></a></p>
                          

                          <p> This is a paragraph. <a id=""></a></p>
                          

                          <q>This is a quotation. </q> 

                          <figure> 
                            <img src="../path/to/your/figure.jpg"> 
                            <figcaption> This is the figure caption. </figcaption> 
                          </figure>
                        </section>

                        <section>
                          <p> This is a paragraph. </p>

                          <p> This is a paragraph. </p>
      
                          <figure>
                            <video src="../path/to/your/video.jpg"> 
                            <figcaption> This is the figure caption. </figcaption>
                          </figure>
                        </section>

                        <section>
                          <p> This is a paragraph. </p>

                          <p> This is a paragraph. </p>

                          <figure>
                            <video src="../path/to/your/video.jpg">
                            <figcaption> This is the figure caption. </figcaption>
                          </figure>
                        </section>

                        <section>
                          <p> This is a paragraph. </p>

                          <figure>
                            <table> 
                              <caption>This is the table caption. </caption> 
                              <tr> 
                                <th>This is a table header</th> 
                                <th>This is another table header</th>
                              </tr>
                              <tr>
                                <td>These are table data in a cell</td> 
                                <td>This is another cell</td>
                              </tr>
                            </table>
                          </figure>

                        </section>

                        <section>

                          <figure>
                            <table>
                              <tr>
                                <td rowspan="">#60;/td>
                                <td colspan="">#60;/td>
                                <td>#60;/td>
                                
                              </tr>
                              <tr>
                                <td>#60;/td>
                                <td>#60;/td>
                              </tr>
                             </table>

                          <ol> 
                            <li>This is the first item. </li> 
                            <li>This is the second item. </li>
                            <li>This is the third item. </li>
                          </ol>

                          <ul> 
                            <li>This is one item of the list. </li>
                            <li>This is another item of the list. </li>
                          </ul>

                        </section>
                      </main>

                      <footer>
                        <p>Here you can put the copyrights. </p>
                      </footer>
                      

                    </body>
