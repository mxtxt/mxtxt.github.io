Widget

Multiple versions of a widget were developed. Before the iframe was implemented, we created a css-themer widget that changed the css stylesheet on the whole current webpage. The themer worked with six buttons that changed the href attribute of the link to the stylesheet of the webpage. This link was identified via its id="original" in the html document. 

          
            <link id=original rel="stylesheet" type="text/css" href="css/style.css">

The six buttons in the html document had the form: 

          
            <button id="theme2" type="button" class="btn btn-light" onclick="theme2()">theme 2</button>

and where controlled by the following javascript:

          
            $(document).ready(function(){
            $("button#theme2").click(function(){
              $("#original").attr("href", "css/style2.css");
              });
            });
